# SellyAnalyseBot

Telegram KI-Verkaufsassistent (Premium)

## Business Analyzer Mini-App

Eine statische HTML-Seite (`index.html`) enthält die Business Analyzer Pro Demo mit Tabs, Slidern und Chart.js-Visualisierung. Öffne die Datei lokal oder mit einem einfachen HTTP-Server, z. B.:

```bash
python -m http.server 8000
```

Rufe dann `http://localhost:8000/index.html` im Browser auf. OAuth, Firebase und Telegram-spezifische Funktionen erfordern eigene Projekt-Keys und die Ausführung in einer Telegram Mini App Umgebung.
